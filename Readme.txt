=====================================
=============操作方法================
=====================================
・キーボードを使用してください

<共通処理>
[←→キー]
	左右移動

[Zキー]
	ジャンプ


<空中にいるとき>
[Zキー]
	上に飛ぶ

[Zキー+←]
	左に飛ぶ

[Zキー+→]
	右に飛ぶ

[Zキー+↓]
	下に飛ぶ

=====================================
============ゲームルール=============
=====================================
・迫ってくる棘から逃げてゴールの旗を目指す。
・コインを獲得するとスコアが増える。
